Reset Local Database

Use this tool to reset your local database and repopulate it with default content.

Database has been reset successfully! Go to website

Instructions

  1. Click the "Reset Local Database" button above
  2. The localStorage will be cleared except for authentication and language settings
  3. The "dbInitialized" flag will be removed
  4. Visit http://localhost:3000 after resetting
  5. The database will be reinitialized with all default content on page load

Important: Make sure your local development server is running (npm run dev).
